# Установка и подключение

Адаптер подключается к прикладной конфигурации в одном из двух режимов: **как расширение** или **как часть основной конфигурации**. В любом случае внешняя компонента для работы с Kafka встроена в подсистему — отдельная установка не требуется.

## Сравнение вариантов

| | Расширение конфигурации | Часть основной конфигурации |
|---|---|---|
| **Рекомендуется для** | Типовых конфигураций с регулярными обновлениями | Глубоко кастомизированных решений |
| **Плюсы** | Минимальное вмешательство в основную конфигурацию; простое отключение и удаление | Полный контроль над точками встраивания; глубокая интеграция в прикладную логику |
| **Минусы** | Ограничения механизма расширений 1С | Сложность обновления зависит от режима поддержки |
| **Индексы СУБД** | [Создаются вручную](https://its.1c.ru/db/v8326doc#bookmark:dev:TI000002802) (автоматически — при лицензии «КОРП») | Создаются автоматически |

После подключения перейдите к [Настройке подсистемы](../configuration/index.md).
