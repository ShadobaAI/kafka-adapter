# Требования

Требования к окружению перед подключением адаптера.

## :material-server: Платформа 1С:Предприятие

- **8.3.21 и выше** — для Windows;
- **8.3.24 и выше** — для Linux.

Режим информационной базы — **только клиент-серверный**. Файловый режим не поддерживается.

Лицензия — **«ПРОФ» и выше**. Для автоматического создания дополнительных индексов потребуется **«КОРП»**.

## :material-library: Библиотека стандартных подсистем (БСП)

**3.1.10 и выше** — обязательное условие для конфигурации, в которую встраивается адаптер.

## :material-apache-kafka: Apache Kafka

- Доступный кластер Apache Kafka.
- Настроенные топики для исходящих и входящих сообщений.

## :material-swap-horizontal: Конвертация данных

При необходимости — XDTO-пакет и модуль обмена [1С:Конвертация данных 3.1](http://its.1c.ru/db/metod8dev#content:5846:hdoc).

## :material-toolbox: Инструменты разработки

Для подключения адаптера потребуется **EDT** или **Конфигуратор** 1С.

## :material-earth-plus: Внешние сервисы (опционально)

Для централизованного логирования истории обмена — любой HTTP/HTTPS-эндпоинт, принимающий JSON:

- [Logstash](https://www.elastic.co/logstash);
- [Elasticsearch](https://www.elastic.co/elasticsearch);
- [Loki](https://grafana.com/oss/loki/);
- любой собственный приёмник JSON.

!!! info "Внешняя компонента"
    Внешняя компонента для работы с Kafka (`Simple Kafka Connector 1C`) **встроена в подсистему**. Отдельная установка и регистрация не требуются.

## :material-chip: Операционная система сервера 1С

- **Windows 64-bit**;
- **Linux 64-bit**.
