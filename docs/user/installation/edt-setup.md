# Настройка в EDT / Конфигураторе

Действия, которые необходимо выполнить на уровне метаданных — **один раз**, при первом подключении адаптера к целевой конфигурации.

## Определяемый тип `кфкПользователь`

В адаптере определён тип `кфкПользователь` для хранения ссылок на пользователей. По умолчанию он указывает на `СправочникСсылка.Пользователи` — служебный справочник из состава адаптера, предназначенный **только для разработки**.

!!! warning "Обязательно заменить"
    При внедрении в прикладную конфигурацию **замените** тип на `СправочникСсылка.Пользователи` **целевой** конфигурации (обычно он есть в БСП — `Справочник.Пользователи`).

### Как заменить

=== "EDT"

    1. Откройте **Определяемые типы** → `кфкПользователь`.
    2. В списке типов удалите `СправочникСсылка.Пользователи` из состава адаптера.
    3. Добавьте `СправочникСсылка.Пользователи` целевой конфигурации.
    4. Сохраните проект.
    5. Обновите конфигурацию базы данных.

=== "Конфигуратор"

    1. Откройте дерево конфигурации → **Общие → Определяемые типы → `кфкПользователь`**.
    2. Двойным щелчком откройте тип.
    3. В окне «Тип данных» измените ссылку `СправочникСсылка.Пользователи` на справочник пользователей целевой конфигурации.
    4. Обновите конфигурацию базы данных.
