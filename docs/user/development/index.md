# Разработка с адаптером

Раздел для 1С-разработчиков, пишущих код прикладного решения — обработчики, регистрации, прямые вызовы API.

!!! tip "Главный принцип"
    Из прикладного кода следует вызывать **только** модули `кфкИнтеграция` (сервер) и `кфкИнтеграцияКлиент` (клиент). Все остальные модули (с суффиксом `Служебный`) — **внутренние**, их сигнатуры могут меняться между версиями.
