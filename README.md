Встраиваемая подсистема для двустороннего обмена между **1С:Предприятие** и **Apache Kafka** — на основе внешнего компонента [Simple Kafka Connector 1C](https://github.com/NuclearAPK/Simple-Kafka_Adapter).

## [Документация →](https://shadobaai.github.io/kafka-adapter/)

Полное руководство, примеры и справочник API доступны на сайте документации.

## Возможности

**1С → Kafka**
- Автоматическая постановка объектов в очередь при записи (через подписки на события)
- Ручная регистрация через UI или программный API
- Произвольные обработчики сериализации или 1С:Конвертация данных 3.1
- Параллельная отправка через фоновые задания

**Kafka → 1С**
- Автоматическая загрузка сообщений из топиков
- Произвольные обработчики десериализации или 1С:Конвертация данных 3.1
- Параллельная обработка через фоновые задания

**Операционность**
- Хранение истории обмена, диагностика статусов
- Алерты в Telegram при нарушении SLA
- Выгрузка журнала обмена в ELK / Loki

## Требования

| | |
|---|---|
| **Платформа** | 1С:Предприятие 8.3.21+ (Linux: 8.3.24+) |
| **ОС** | Windows 64-bit / Linux 64-bit |
| **БСП** | 1С:Библиотека стандартных подсистем 3.1.10+ |
| **Kafka** | Доступный кластер Apache Kafka |
| **Лицензия 1С** | «ПРОФ» и выше |
